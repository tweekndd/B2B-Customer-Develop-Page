上手

快速开始

几分钟内启动系统，开始你的客户开发之旅。

📦 安装依赖

# 克隆项目 git clone https://github.com/tweekndd/B2B-Customer-Develop-Page.git cd B2B-Customer-Develop-Page # 安装 Python 依赖 pip install -r requirements.txt

🔑 配置环境变量

# 必须设置管理员账号 ADMIN_USERNAME=admin ADMIN_PASSWORD=your-secure-password GLM_API_KEY=your-glm-api-key SERPAPI_API_KEY=your-serpapi-key # 或 TAVILY_API_KEY # 可选：SearXNG 自托管（零成本） SEARXNG_URL=http://127.0.0.1:8888

🚀 启动系统

# 一键启动 python main.py # 浏览器访问 → http://localhost:8000 # 使用管理员账号登录 用户名: admin 密码: your-secure-password

💡 零成本搜索推荐： 部署 SearXNG Docker 容器，彻底告别 API 费用。 详见 README 中的 部署指南。

🐍 Python 3.10+ 📦 依赖一键安装 🔐 管理员自动创建 🌐 开箱即用
部署

生产环境一键部署

支持 Docker 容器化部署，搭配 SearXNG 自托管搜索引擎，零 API 成本。

🐳 推荐

Docker 部署

在 VPS 上使用 Docker Compose 一键启动所有服务：应用 + SearXNG + 可选 PostgreSQL。

# 克隆项目并启动 git clone https://github.com/tweekndd/B2B-Customer-Develop-Page.git cd B2B-Customer-Develop-Page bash deploy.sh
  • 自动构建 Docker 镜像
  • 启动 SearXNG 搜索引擎（内网）
  • 启动 FastAPI 应用（端口 8000）
  • 支持 PostgreSQL 生产数据库
🌐 生产就绪

Nginx + HTTPS

配置 Nginx 反向代理 + Let's Encrypt 免费 SSL 证书，安全访问你的系统。

# 安装 Nginx 并配置反代 apt install nginx -y # 配置 /etc/nginx/sites-available/b2b certbot --nginx -d your-domain.com
  • 域名绑定 + SSL 加密
  • 静态资源缓存优化
  • 负载均衡支持（多实例）
🧩 服务组件
📦 b2b-app :8000 🔍 searxng :8888 (内网) 🗄️ postgres :5432 (可选)
docker compose up -d bash deploy.sh update

项目结构

清晰的分层架构，便于二次开发与维护。

AI-Trade-Customer-Analyzer/
├── main.py # FastAPI 主入口 (V4.1)
├── Dockerfile # Docker 镜像构建
├── docker-compose.yml # 编排：app + searxng + db
├── deploy.sh # 一键部署脚本
├── .env.example # 环境变量模板
├── requirements.txt # Python 依赖
├── app/
│ ├── database.py # 13 张表 (含 User 表)
│ ├── auth.py # 认证与权限 (V4.1)
│ ├── api/ # 路由：customers / discovery / users / hunter ...
│ ├── services/ # 核心服务：爬虫 / 分析 / 搜索 / 邮箱 / 缓存 ...
│ ├── static/ # JS + CSS
│ └── templates/ # HTML 模板 (登录 / 列表 / 详情 / 地图 ...)
└── searxng/ # SearXNG 配置 (启用 JSON API)
